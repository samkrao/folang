## Application

    Folang Application is a standalone executable binary
   
        Which consists of
   
            1. Code block (Entry Point one and only one)
            2. Packages


## Library

   Folang Library is a reusable Collection of utilities which needs to be integrated with other Libraries or Application

        Which consists of

            1. Library which can spawn into multiple files
                 
                 1. Packages

                    Annotated with @co.dap.library

                    if package contains structs and free functions
                    their declarations must be put in signatures

    
    
   Please refer [Libaray](Library.md) for more details
