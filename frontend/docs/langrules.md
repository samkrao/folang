# co-lang Programming

## Language is similar to C. 
    Semi colon is terminator  
    Comma is separator
    Braces for Block start and Block End

#### 1. A co lang source file can contain 
    a. Only one package declaration or
    b. Bunch of code statements

#### 3.  Code statements are
    a. Importing packages and/or modules,
    b. Simple expressions,
    c. Declaring variables,
    c. Function calls,
    d. Conditional checks and loops
    e. annotation declarations

#### 4. A package can contain
    a. Packages
    b. Module
    c. Classes
    d. Functions
    e. Types
    f. annotation declarations
    g. annotation definitions
    h. macros
    i. templates
    j. generics and parametrics fuctions only
    k. Many others

#### 5. Module contain
    a. Modules,
    b. Types, and
    c. Functions 
    d. annotation declarations

#### 6. Types can compose type like (golang)

#### 7. Functions are curried/asm/machinecode/normal/clousure contain 
    a. Statements
    b. Expressions
    c. Functions
    d. Anyonymous/named funtions/blocks
    e. Lambda expressions and functions
    f. annotation declarations
    g. blocks
    h. Continuations callsite / delimited
    i. threads/tasks/processes, coroutines, channels, fibers, asyc/await calls etc
    j. parametric, generic and variances(co, contra, in)
    
   
#### 8. Classes can contain
    a. Classes
    b. Methods
    c. Variable Declarations
    d. annotation declarations
    e. delegates
